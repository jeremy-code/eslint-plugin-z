## Installation

```shell
git clone https://github.com/jeremy-code/eslint-plugin-z.git
cd eslint-plugin-z
corepack enable
pnpm install
```

## Usage

```shell
# Set Node.js version to the one specified in .nvmrc
nvm use

# Lint and format code with ESLint and Prettier
pnpm lint
pnpm format

# Build
pnpm build

# Run unit tests with Jest
pnpm test
pnpm test:watch

```
